Telegram 中泰翻譯機器人 PRO

1. 把 .env.example 複製成 .env
2. 填入 TELEGRAM_BOT_TOKEN / OPENAI_API_KEY / OWNER_TELEGRAM_ID / RENDER_EXTERNAL_URL
3. 安裝套件：npm install
4. 啟動：npm start
5. 到 BotFather 設定你的 bot：
   - /setprivacy -> Disable
6. 把 bot 加入群組
7. 在群組輸入 /authorize（僅 OWNER_TELEGRAM_ID）
8. 設定模式：/mode auto

常用指令：
/start
/help
/myid
/chatid
/status
/on
/off
/mode auto
/mode zh2th
/mode th2zh
/mode manual
/settarget zh
/settarget th
/showoriginal on
/showoriginal off
/translate 你好
